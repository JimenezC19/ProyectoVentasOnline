# Dashboard de Ventas del Canal Online

## Descripción
Dashboard de Business Intelligence desarrollado en Power BI para monitorear y analizar el desempeño de ventas del canal online. La solución proporciona visibilidad sobre indicadores clave de ventas, comportamiento de pedidos, tendencias comerciales y desempeño de canales digitales, facilitando el seguimiento operativo y la toma de decisiones estratégicas orientadas al crecimiento comercial.


## Objetivo del Proyecto
Desarrollar un dashboard de Business Intelligence en Power BI para monitorear y analizar el desempeño de ventas del canal online, proporcionando visibilidad sobre KPIs comerciales, comportamiento de ventas y tendencias digitales. La solución permite facilitar el seguimiento operativo y apoyar la toma de decisiones estratégicas orientadas al crecimiento comercial y optimización del canal digital.


## KPIs Principales
- Ventas totales online
- Pedidos realizados
- Ticket promedio
- Crecimiento de ventas
- Productos más vendidos
- Ventas por canal digital
- Tendencias semanales, mensuales y anuales
- Desempeño comercial por categoría de producto


## Funcionalidades Principales
- Monitoreo de ventas y desempeño del canal online.
- Seguimiento de KPIs comerciales y métricas de ventas.
- Análisis de pedidos y comportamiento de compra.
- Visualización de tendencias semanales, mensuales y anuales.
- Análisis de desempeño por canal digital y categoría de producto.
- Identificación de productos con mayor volumen de ventas.
- Comparativos de desempeño comercial entre periodos.
- Filtros interactivos y capacidad de drill-down para análisis detallado.
- Visualización ejecutiva orientada a la toma de decisiones.


## Arquitectura de Datos

```plaintext
Base de Datos SQL
        ↓
Consultas SQL
        ↓
Procesamiento y automatización con Python
        ↓
Google BigQuery
        ↓
Modelo Semántico en Power BI
        ↓
Dashboard y actualización programada
```

## Proceso ETL
1. Extracción de datos desde una base de datos SQL
2. Transformación y carga automatizada mediante scripts en Python
3. Almacenamiento de información en Google BigQuery
4. Conexión del modelo semántico de Power BI al entorno cloud
5. Actualización programada para monitoreo continuo de KPIs


## Herramientas Utilizadas
- Power BI
- Power Query
- DAX
- Modelado de Datos
- Python
- SQL
- Google BigQuery

## Insights Clave
- Identificación de periodos con mayor volumen de ventas online.
- Detección de canales digitales con mejor desempeño comercial.
- Variaciones relevantes en el ticket promedio entre categorías y periodos.
- Identificación de productos con mayor participación en ventas.
- Monitoreo de tendencias de crecimiento del canal digital.
- Mayor visibilidad del comportamiento comercial mediante KPIs centralizados e interactivos.


## Valor de Negocio
Este dashboard permite monitorear y centralizar los principales indicadores de desempeño del canal online, proporcionando visibilidad sobre ventas, comportamiento de pedidos y tendencias comerciales. La solución facilita la toma de decisiones estratégicas, el seguimiento operativo y la identificación de oportunidades de crecimiento mediante análisis interactivos y KPIs orientados al negocio.
